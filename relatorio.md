 Introdução: Esse trabalho foi realizado para a disciplina estrutura de dados com a proposta de desenvolver a parte prática que foi estudada no semestre, usando a linguagem C para utilizar diferentes estruturas de dados para resolver problemas do cotidiano. 


Objetivos:

Implementar estruturas de dados em linguagem C.
Compreender o funcionamento de listas, pilhas e filas.
Desenvolver programas com operações de inserção, remoção, busca e edição de dados.
Aplicar conceitos de manipulação de arquivos CSV.


Desenvolvimento:

Trabalho A – Playlist de Músicas: foi implementada uma lista duplamente encadeada para gerenciar músicas, podendo cadastrar, editar, remover, buscar, listar, salvar e carregar outras informações. 

Trabalho B – Histórico de Navegação Web: foi usada uma pilha dinâmica para reproduzir o histórico de um navegador, permitindo visitar páginas, voltar, avançar, pesquisar, editar, excluir e salvar os dados. 

Trabalho C – Fila de Impressão: foi criada uma fila dinâmica para simular um sistema de impressão, possibilitando adicionar documentos, processar impressões, cancelar trabalhos, realizar buscas e armazenar os dados em arquivos CSV. 

Resultados: Os três programas funcionaram conforme o esperado e atenderam aos objetivos do trabalho. durante os testes, foi possível realizar todas as funções planejadas para cada estrutura de dados. além disso, foram adicionados alguns recursos extras, como geração de estatísticas, opções de pesquisa e salvamento das informações em arquivos CSV.


Dificuldades Encontradas:Durante o decorrer do trabalho, não tivemos grandes dificuldades, algumas partes exigiram mais atenção, mas conseguimos resolver os problemas ao longo da implementação e finalizar todas as funcionalidades previstas para o projeto.

Conclusão:Com a realização deste trabalho, conseguimos colocar em prática o que foi aprendido no decorrer do semestre da disciplina Estrutura de Dados. o desenvolvimento dos programas ajudou a entender melhor o funcionamento das listas, pilhas e filas, além de proporcionar mais experiência com programação em C e trabalho em equipe. no final, o projeto contribuiu para reforçar os conhecimentos da disciplina e mostrar como essas estruturas são utilizadas na prática.
